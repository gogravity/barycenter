# Microsoft Azure BAA Reference

Microsoft executes a single BAA covering all HIPAA-eligible Azure services for customers
accepting the Online Services Terms (OST). No separate signature is required; the BAA is
auto-applied via the OST.

**Reference:** https://www.microsoft.com/licensing/docs/view/Online-Services-Terms (HIPAA BAA section)
**Service Trust Portal:** https://servicetrust.microsoft.com/

**HIPAA-eligible services Barycenter relies on:**
- Azure SQL Database
- Azure Storage (Blob, immutable containers)
- Azure Key Vault (Standard tier; Premium/Managed HSM upgrade path preserved)
- Azure Monitor / Log Analytics
- Microsoft Entra ID (P2 — for PIM)
- Azure Virtual Network / Private Endpoints

**Verification:** Subscription owner attests via Service Trust Portal annual review.
